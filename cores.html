Como construir uma Interface gráfica?

Combinações de cores em uma interface gráfica


  • Umas das maiores preocupações quando se cria uma janela web, são as
    combinações das cores.
  • Este tópico é muito importante porque as cores da janela é uma das primeiras
    coisas que o utilizador vê e analisa.

  • Para se obter uma boa combinação de cores tem de ser ter atenção a alguns
    parâmetros:

  • As classes das cores: Primárias, Secundárias ou Terciárias
  • Ser Quente ou Fria
  • Ser Complementar ou Análoga.
