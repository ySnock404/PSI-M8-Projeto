Como construir uma Interface gráfica?

Regras na construção de uma interface Gráfica


Quando se está a criar uma interface gráfica, ou Janela web é muito
importante ter em conta alguns pormenores que podem estragar a experiência
do utilizador:

  • Certificar que as frases são curtas e simples o
    para não afastar o utilizador
  • Ter em conta que as frases são positivas
  • Ordenar os dados de forma lógica ou alfabética
  • Ter muita atenção às combinações de cores utilizadas.
